# Parkinson’s Early Detection using XGBoost

## Project Description
This project implements an intelligent system for early detection of Parkinson’s Disease using biomedical voice measurements and an optimized XGBoost classifier.

## Dataset
UCI Parkinson’s Disease Dataset  
https://archive.ics.uci.edu/ml/datasets/parkinsons

## Methodology
- Advanced data preprocessing
- XGBoost model development
- Hyperparameter optimization using GridSearchCV

## Deployment
The trained model is deployed as a web application using Streamlit Cloud.

## How to Run Locally
pip install -r requirements.txt  
streamlit run app.py
